Quick Start Guide

Follow these steps to begin your programming journey with Python:

  1. Install Python: Go to python.org/downloads
  2. Download the latest version for your operating system.
  3. On Windows, check “Add Python to PATH” before clicking Install.
  4. Write Your First Script: Open IDLE or VS Code. Type the following line and run it.
    5. print("Hello, World!")
    1. Save Your File: Save your script with a clear name like hello.py.
    2. Lern Basic Concepts:
      • Variables
      name = "Canaan"
age = 18
print(name, age)
      • Input and Output
      name = input("Enter your name: ")
print("Welcome", name)
      • If Statements
      if age > 18:
    print("Adult")
else:
    print("Underage")
      • Loops
      for i in range(5):
    print(i)
    3. Practice Small Problems Daily
      • Write small scripts that calculate or repeat tasks.
      • Use w3schools.com/python or programiz.com/python.
      • Practice on Replit or LeetCode beginner problems.
    4. Save and Organize Your Code
      • Create a folder named MyPythonProjects.
      • Save each file with a clear name:
        • hello.py
        • calculator.py
        • loop_practice.py
    5. Move Forward

      6. After learning the basics, continue with:

      • Functions
      • Lists and Dictionaries
      • File Handling
      • Error Handling (try/except)
      • Simple projects like a calculator or quiz app
